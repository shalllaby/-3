## 📖 Business Vision

**Anhar Al-Deera (أنهار الديرة)** is a next-generation B2C e-commerce platform specifically engineered for the unique dynamics of the Kuwaiti retail landscape. Unlike generic solutions, Anhar Al-Deera utilizes the latest modern web technologies, combined with **Gemini Flash** integration, to provide a hyper-localized, intelligent shopping experience.

The platform bridges the gap between traditional retail and modern digital requirements by offering:
- **Ultra-Fast Logistics:** Integration with all local Kuwaiti governorates for delivery within 30-150 minutes, tracking through tailored shipping models.
- **AI-Driven Edge:** Experimental **Gemini Flash** integration for dynamic content generation and Arabic product optimizations.
- **Financial Localization:** Support for KNET, Pay Deema, Apple Pay, Visa/MasterCard, and Cash on Delivery (COD) in Kuwaiti Dinar (KWD).
- **RTL-First Design:** A premium user interface designed natively for Arabic speakers using Next.js 14 and Tailwind CSS.
- **Global Reach:** Automatic translation support for 35+ world languages.
- **Referral & Loyalty Programs:** Built-in referral codes, user coupons, and recurring package management (Home, Work, Custom).

## 🏗️ Technical Architecture

The project employs a robust **Monorepo** architecture leveraging `pnpm workspaces` to separate concerns seamlessly between the Database, API gateway, and Client-facing storefront.

```mermaid
graph TD
    User((User)) --> Web[Next.js 14 Frontend - Port 3000]
    Web --> API[NestJS Gateway - Port 3001]
    Web --> Trans[Google Translate Engine - 35+ Langs]
    API --> Gemini[Gemini Flash AI Integration]
    API --> DB[(PostgreSQL + Prisma)]
    API --> Auth[Passport & JWT]
    
    subgraph Frontend Stack
        Zustand[Zustand State]
        RQ[React Query]
        Lucide[Lucide Icons]
    end
    Web -.-> Frontend Stack
```

## ⚙️ Workflow & Internal Logic

### 1. Unified Development Flow
Due to recent modifications resolving `EADDRINUSE` port bottlenecks, engineers now just execute `pnpm dev` at the root. The `concurrently` package gracefully stages both the Next.js storefront and the NestJS REST backend simultaneously.

### 2. The Transaction Journey
1. **Catalog Browsing:** Visitors navigate dynamically built category/offer pages. Data is consumed quickly using React Query and Zustand stores.
2. **Account Creation:** The backend hashes identities with `bcrypt` while `JWT` grants session tokens. A unique `referralCode` is generated dynamically matching Prisma schemas.
3. **Checkout Validation:** The client forms (`react-hook-form` + `zod`) align addresses strictly to the 6 Kuwait Governorates. 
4. **Order Confirmation:** Orders transition to `PENDING` awaiting KNET signals, logging immutable purchase records (`priceAtPurchase`) in `order_items`.

## 🔌 External Integrations & Data Layer

- **Database:** Fully relational PostgreSQL. Self-relations on Categories, JSON/Text scaling for delivery notes, and complex relations linking Coupons, Packages, and Referrals.
- **AI API:** **Gemini Flash** model hooked directly into operational pipelines to manage text extraction and high-volume description rewriting.
- **REST Validation:** Native `class-validator` and `class-transformer` pipes sanitize all inbound JSON from Next.js before writing to Postgres.
- **API Documentation:** Automatic OpenAPI documentation dynamically generated by `@nestjs/swagger` available at backend boot.

## 🚀 Setup & Execution Instructions

### Prerequisites
- Node.js v19+ (Recommend v20.x)
- pnpm package manager (`npm -g i pnpm`)
- PostgreSQL Database (local or cloud)

### Configuration
1.  **Dependencies Install:**
    Navigate to the root `متجر-الكويت-2` folder and run:
    ```bash
    pnpm install
    ```

2.  **Environment Settings:**
    Database configs must be applied at `packages/database/.env`:
    ```env
    DATABASE_URL="postgresql://username:password@localhost:5432/anhar_aldeera"
    ```
    Backend configs go to `apps/api/.env`:
    ```env
    JWT_SECRET="YOUR_SUPER_SECRET_KEY"
    GEMINI_API_KEY="YOUR_GEMINI_API_SECRET"
    ```

3.  **Database Migration & Seeding:**
    Push the new Prisma structure (which includes recent referral modifications) to your database:
    ```bash
    pnpm db:generate
    pnpm db:migrate
    pnpm db:seed
    ```

### Running Locally
To launch both the Next.js Frontend and NestJS API at once, use the unified script:
```bash
pnpm dev
```
- **Next.js E-Commerce Interface:** `http://localhost:3000`
- **NestJS API Back-end:** `http://localhost:3001`
- **API Swagger Docs:** `http://localhost:3001/api/docs`
- **Prisma Studio:** `pnpm db:studio` (Runs at port 5555)
